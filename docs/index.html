About davinci-pic

The Davinci-Pic web component serves as a seamless extension of the DavinciPics API. Its primary function is to streamline the process of fetching and displaying visual identities associated with Hedera entities. Not only does it make it incredibly straightforward to pull this information, but it also offers a plethora of customization options to ensure that the visual elements align with the specific needs of your application.

Supported Entity Types

As you may already know, the DavinciPics API provides support for six distinct types of Hedera entities:

  • Tokens
  • Profile Pictures
  • Banners
  • Nodes
  • Networks
  • Apps

Each of these entity types has unique visual identifiers, and our Davinci-Pic component is designed to handle all of them adeptly.
